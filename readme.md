NextGuard

누가 필요한가?
- 산업 안전 관리자

왜?
- 산업 안전 관리자가 여러 공장 업체를 돌아다니면서, 산업 안전 법에 위반하는 현장의 모습을 찍고, 왜 부적절한지 그 이유를 작성하고 그와 관련된 법률을 관리자 스스로가 작성해야한다.
- 이런 번거로운 작업을 회피하고 더 정확한 시스템을 구축하기 위해서 AI Agent 시스템을 도입한다.

언제?
- 산업 안전 관리자가 업무를 볼 때 필요

무엇을?
- 산업 안전 처벌법 방지를 위한 AI Agent 시스템

어떻게 개발?
- AI Gen Image로 부적절한 산업 재해 사진을 적합하게 변경 
- RAG 시스템 (법 조항 근거 검색) 
- MCP 통신을 이용해서 외부 소스 참고 
- LangGraph로 부적한 사항을 조금 더 고도화해서 작성할 수 있도록 AI Agent 구현 (작성자 - 검토자 Workflow를 구현하여 반복 수행 후 기준이 통과되면 출력)

요구 사항
1. 점검결과 보고서는 HWPX 문서 형태로
2. 현장에서 점검 사진과 부적합 사항 내용을 점검자가 작성하여 업로딩.
3. 업로딩 되면, AI(제미나이 2.5)가 점검 사진과 부적합 사항을 자동 분석.(자동분석은 25년 법제처 중대재해처벌법, 산업안전보건법, 산업안전보건기준 규칙, 고용노동부 고시 범위에서 분석)
4. 자동 분석을 통해 개선 사항 내용과 개선된 사진 및 그에 따른 법령근거(조항 및 세부내용)를 해당되는 작성란에 자동 업로딩.
